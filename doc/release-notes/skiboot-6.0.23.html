skiboot 6.0.23 was released on Tuesday April 7th, 2020. It replaces skiboot-6.0.22 as the current stable release in the 6.0.x series.

It is recommended that 6.0.23 be used instead of any previous 6.0.x version due to the bug fixes it contains.

Bug fixes included in this release are:

  • npu2: Clear fence on all bricks

  • npu2: Clear fence state for a brick being reset
